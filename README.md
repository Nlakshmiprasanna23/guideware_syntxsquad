# 📌 Problem Statement

Food delivery partners face frequent income loss due to external disruptions such as:

- Heavy Rain 🌧  
- Extreme Heat ☀  
- Government Curfews 🚫  
- Platform Downtime 📱  

Currently, there is **no structured system** to protect their **weekly income**.

---

# 👤 Delivery Persona

**Name:** Rahul  
**Role:** Food Delivery Partner (Swiggy/Zomato)  
**Location:** Hyderabad  

### Profile:
- Works 8–10 hours daily  
- Weekly income: ₹6000–₹8000  
- Fully dependent on weather and app availability  

---

# 💡 Proposed Solution

We propose an **AI-powered parametric insurance platform** that:

- Offers **weekly income protection**
- Uses **AI risk scoring** for premium calculation
- Detects disruptions automatically via triggers
- Processes claims instantly without manual intervention

---

# 💰 Weekly Premium Model

Premium is calculated using an AI Risk Score.

### Formula:
Premium = ₹20 + (Risk Score × ₹15)

### Example:
- Risk Score = 0.6  
- Premium = ₹29/week  

---

# ⚡ Parametric Triggers

Claims are automatically triggered when:

- Rainfall exceeds threshold 🌧  
- Temperature exceeds 42°C ☀  
- Government curfew is declared 🚫  
- Platform downtime occurs 📱  

---

# 🤖 AI Usage

AI is used in:

1. Risk Prediction  
2. Dynamic Premium Calculation  
3. Fraud Detection (planned for future phases)

# 🏗 Tech Stack

- Frontend: HTML, CSS, JavaScript

# 🔁 Workflow

1. User logs in  
2. AI calculates risk score  
3. Weekly premium is generated  
4. User activates policy  
5. System monitors external APIs  
6. Disruption trigger is detected  
7. Claim is automatically initiated  
8. Fraud check is performed  
9. Instant payout is process
